## Projects

### [DealCrunch](https://github.com/rizwanpirzada/dealcrunch)
Real-estate deal underwriting tool. Replaces spreadsheets with a proper 10-year proforma engine — enter a property, dial in assumptions, compare scenarios side by side, share a read-only link.

IRR is computed with Newton-Raphson + bisection fallback (no library). All money stored in integer cents to prevent floating-point drift over 30-year projections. The proforma row generator is lazy so sensitivity analysis never holds 100 full models in memory at once.

`React + Vite` `Node/Express` `TypeScript` `SQLite` `Zod` `JWT` `Vitest`

---

### [Chicago Bus System](https://github.com/rizwanpirzada/chicago_bus_system)
